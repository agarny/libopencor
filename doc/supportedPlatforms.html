libOpenCOR can be used on the following versions of Windows, Linux and macOS. Alternatively, you can try building it yourself.

Windows
  • Snapshots: supported on the 64-bit version of Windows 10.
Linux
  • Snapshots: supported on the 64-bit version of Ubuntu 20.04 LTS (Focal Fossa) and later.
macOS
  • Snapshots: supported on macOS 10.14 (Mojave) and later.
